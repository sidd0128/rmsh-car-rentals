# Internationalization (i18n)

Same approach as **Avy**: [`i18n-js`](https://github.com/fnando/i18n-js) + JSON locale files under `src/locales/`.

## Layout

| Path | Purpose |
|------|---------|
| `src/locales/en.json` | English copy (nested keys) |
| `src/locales/sv.json` | *(future)* second locale |
| `src/core/i18n/index.ts` | `I18n` instance + `t` / `useTranslation` |

## Setup (matches Avy)

```ts
import { I18n } from 'i18n-js';
import en from '@locales/en.json';

const i18n = new I18n({ en });
i18n.enableFallback = true;
i18n.defaultLocale = 'en';
i18n.locale = 'en';
```

## Usage in components

```tsx
import { useTranslation } from '@core/i18n';

const { t } = useTranslation();
return <Text>{t('dashboard.title')}</Text>;
```

Or like Avy, import the default instance:

```tsx
import i18n from '@core/i18n';

<Text>{i18n.t('dashboard.title')}</Text>
```

## Outside React

```ts
import i18n from '@core/i18n';
// or
import { t } from '@core/i18n';

i18n.t('navigation.fleet');
t('dashboard.returnsSoonDescription', { days: 3 });
```

Interpolation in `en.json` uses `{{variable}}` (configured in `src/core/i18n/index.ts`).

## Adding Swedish (example)

1. Add `src/locales/sv.json`.
2. Register in `src/core/i18n/index.ts`:

```ts
import sv from '@locales/sv.json';

const i18n = new I18n({ en, sv });
```

3. Switch locale: `i18n.locale = 'sv'` and re-render (e.g. `i18n.onChange` + state in a settings screen).

## Bootstrap

`import '@core/i18n'` in `App.tsx` and `jest.setup.ts`.
